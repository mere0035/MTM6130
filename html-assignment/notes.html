Notes

<del>
Del, also known as delete is used to represent text that has been deleted. Example
<div>
Div or a division tag is used to as a container to section HTML elements. A class or id tag can be used to style with a CSS page.
<!DOCTYPE>
Doctype is an HTML declaration which is required in every HTML document to inform the web browser as to what version of HTML is being used.
<textarea>
Textarea tag is used to create a text box for users to input multiple lines of text, this is typically used in a form.
<ul>
Ul, also known as unordered list is used to create a list with bullet points. The bullet point appearance can re-styled with CSS.
